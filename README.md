## docs.pioreactor

#### Install npm

https://nodejs.org/en/download/

### Clone this repo:

```
git clone git@github.com:Pioreactor/docs.pioreactor.git
```

#### Start local dev server

```
npx docusaurus start
```


#### Deploy
```
USE_SSH=true GIT_USER=<your github username>  npx docusaurus deploy
```
